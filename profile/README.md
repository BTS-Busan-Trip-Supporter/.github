## 📄 프로젝트 소개
'P의 여행'이란 부산지역의 근거리 여행 활성화를 위해 개발된 지역 맞춤형 서비스입니다. <br/>
다녀온 여행지와 주변 명소 등을 간단한 방식으로 평가하고 <br/>
여러가지 미션을 통해 아이템을 획득하여 여행의 즐거움을 제공합니다.
